	The following function returns `true` if the enchantment contains at least one hostile effect. This criterion can be considered subjective, but that rule is also used by the game to determine if use of enchantment on an NPC is considered an offense.

	```lua

	local function isSpellHostile(magicSource)
	    for _, effect in ipairs(magicSource.effects) do
	        if (effect.object.isHarmful) then
				-- If one of the spell's effects is harmful, then
				-- `true` is returned and function ends here.
	            return true
	        end
	    end
		-- If no harmful effect was found then return `false`.
	```
